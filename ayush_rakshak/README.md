# 🏥 InsightVault: Digital Health Archive

InsightVault is an AI-powered digital healthcare platform built with Django. It allows users to securely upload their physical medical reports, extracts the data using AI OCR, and provides intelligent risk analysis and personalized diet plans.

## ✨ Core Features (In Progress)
* **Secure User Authentication:** Custom registration and login system.
* **Modern Dashboard:** Intuitive UI/UX for managing health records.
* **Smart Extraction (Upcoming):** OCR technology to read data from uploaded report images.
* **AI Dietician (Upcoming):** Automated meal plans based on extracted health parameters.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
* **Backend:** Python, Django
* **Database:** SQLite (Development) -> PostgreSQL (Production)

## 🚀 How to Run Locally
1. Clone the repository.
2. Install dependencies (Django).
3. Run `python manage.py makemigrations` and `python manage.py migrate`.
4. Run `python manage.py runserver`.
